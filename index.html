Extensions de navegador

Extensions de canvi d'idioma

Tres extensions — En català, si us plau, Euskaraz, mesedez i En galego, por favor — que permeten redirigir el navegador de manera completament automàtica a la versió de la pàgina web que estiguem visitant que contingui alguna d'aquestes llengües minoritzades. Sense clicks addicionals, sense desplegar menús, sense configurar res a cada web. Pensades per a parlants de català, euskera i gallec que volen navegar per internet plenament en la seva llengua.

Disponibles per a Chrome, Brave, Ecosia (i altres navegadors Chromium) i Firefox. Pròximament per a Edge, Safari i més navegadors.

  • JavaScript
  • WebExtensions API
  • Chrome API
  • i18n

Justícia social

Alerta Desnona

Una aplicació d'alertes de desnonaments a la Península Ibèrica. Amb un mapa interactiu que mostra en temps real les subhastes i desnonaments publicats al BOE, permetent a la ciutadania i les plataformes d'habitatge organitzar-se i actuar davant d'aquestes situacions. En un futur, la idea és fer-ne una eina de denúncia i pressió política per a la reforma de les lleis d'habitatge.

Inclou notificacions push, alertes per correu electrònic i un sistema de filtratge per zona geogràfica.

  • React
  • TypeScript
  • Leaflet
  • Node.js
  • Express
  • SQLite
  • Web Push

Plataforma clínica

Open Data Capture

Plataforma web de codi obert dissenyada per a la recollida contínua de dades clíniques. Permet administrar instruments clínics de forma remota i presencial, amb un sistema flexible de creació d'instruments, visualització de dades i exportació sota demanda.

Contribució a un projecte del Douglas Neuroinformatics Platform. Multilingüe, segur per defecte amb autenticació JWT i permisos granulars.

  • TypeScript
  • React
  • Docker
  • WebAssembly
  • JWT
  • REST API

Repositori fotogràfic

Aparador

Aparador és una plataforma web pensada perquè els fotògrafs puguin compartir àlbums amb el seu públic mitjançant codis d'accés. El públic hi pot navegar amb comoditat, veure les fotos en galeria i descarregar-les en la seva resolució original.

Inclou donatius opcionals via Stripe, dashboard per gestionar àlbums i pujades de fotos, i una arquitectura lleugera per mantenir costos baixos.

  • Next.js
  • TypeScript
  • Supabase
  • Cloudflare R2
  • Stripe

Eina web per a empreses associades

Calculadora del factor solar total per a AESSO

He creat una eina perquè les empreses de l'Associació Espanyola de la Protecció Solar puguin calcular el rendiment d'un sistema de protecció solar en pocs segons. Introdueixen les dades bàsiques, veuen el resultat de seguida i poden descarregar un informe en PDF llest per compartir.

L'eina està integrada al web d'AESSO i només hi poden accedir les empreses associades. Així tenen un procés simple, ràpid i segur, sense dependre de fulls de càlcul ni càlculs manuals.

  • HTML / CSS / JS
  • WordPress Plugin
  • PHP
  • Brevo API
  • HMAC
Plantilla d'enginyeria amb IA

Base Skills Repo

Es tracta d'un repositori base per començar projectes amb GitHub Copilot ben configurat des del primer dia. Organitza el coneixement en capes (instruccions, capacitats, ordres, modes de xat i memòria del repositori) perquè l'agent carregui només el context rellevant a cada tasca.

  • GitHub Copilot
  • Bones pràctiques de codi
  • Agents d'Intel·ligència Artificial
  • Optimització de tokens i stack
