Extensions de navegador

Extensions de canvi d'idioma

Tres extensions — En català, si us plau, Euskaraz, mesedez i En galego, por favor — que permeten redirigir el navegador a la versió de la pàgina web que estiguem visitant que contingui alguna d'aquestes llengües minoritzades amb un sol clic. Pensades per a parlants de català, euskera i gallec que volen navegar per internet plenament en la seva llengua.

Disponibles per a Chrome, Brave, Ecosia (i altres navegadors Chromium), Firefox i Edge. Pròximament per a Safari i més navegadors.

  • JavaScript
  • WebExtensions API
  • Chrome API
  • i18n

Addon de streaming

Stremio en Català

Un addon per a Stremio que ofereix un catàleg complet de pel·lícules i sèries disponibles en català a les principals plataformes d'streaming: 3Cat, Filmin, Netflix, Prime Video, Disney+, Max, Movistar+ i més. Pensat per a tothom que vulgui gaudir de contingut audiovisual en la seva llengua.

  • Node.js
  • Stremio SDK
  • Express
  • TMDB API
  • Vercel

Justícia social

Alerta Desnona

Una aplicació d'alertes de desnonaments a la Península Ibèrica. Amb un mapa interactiu que mostra en temps real les subhastes i desnonaments publicats al BOE, permetent a la ciutadania i les plataformes d'habitatge organitzar-se i actuar davant d'aquestes situacions.

Inclou notificacions push, alertes per correu electrònic i un sistema de filtratge per zona geogràfica.

  • React
  • TypeScript
  • Leaflet
  • Node.js
  • Express
  • SQLite
  • Web Push

Plataforma clínica

Open Data Capture

Plataforma web de codi obert dissenyada per a la recollida contínua de dades clíniques. Permet administrar instruments clínics de forma remota i presencial, amb un sistema flexible de creació d'instruments, visualització de dades i exportació sota demanda.

Contribució a un projecte del Douglas Neuroinformatics Platform. Multilingüe, segur per defecte amb autenticació JWT i permisos granulars.

  • TypeScript
  • React
  • Docker
  • WebAssembly
  • JWT
  • REST API
