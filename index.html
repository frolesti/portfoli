Extensions de navegador

Extensions de canvi d'idioma

Tres extensions — En català, si us plau, Euskaraz, mesedez i En galego, por favor — que permeten redirigir el navegador a la versió de la pàgina web que estiguem visitant que contingui alguna d'aquestes llengües minoritzades amb un sol clic. Pensades per a parlants de català, euskera i gallec que volen navegar per internet plenament en la seva llengua.

Disponibles per a Chrome, Brave, Ecosia (i altres navegadors Chromium), Firefox i Edge. Pròximament per a Safari i més navegadors.

  • JavaScript
  • WebExtensions API
  • Chrome API
  • i18n

En català, si us plau

Chrome / Brave / Ecosia Firefox Edge Safari (pròximament)

Euskaraz, mesedez

Chrome / Brave / Ecosia Firefox (pròximament) Edge Safari (pròximament)

En galego, por favor

Chrome / Brave / Ecosia Firefox (pròximament) Edge Safari (pròximament)
CA EU GL

⚠ Les traduccions a l'euskera i al gallec han estat generades per IA i poden contenir errors.

Automatització

Bot del SEPE

Un bot que automatitza les gestions més tedioses amb el Servei Públic d'Ocupació Estatal (SEPE). Nascut de la frustració compartida per milers de persones que han de navegar per una burocràcia digital opaca i lenta. Aquesta eina facilita processos i estalvia temps i maldecaps.

  • Python
  • Selenium
  • Automatització
  • UX Social

troBar

troBar t'ajuda a trobar bars on veure els partits del FC Barcelona a prop teu, visquis on visquis. Perquè seguir el Barça és millor en companyia, amb un bon ambient i una bona cervesa. Inclou info de partits en temps real, mapa interactiu i filtres per ubicació.

  • React Native
  • Geolocalització
  • API de mapes
  • Firebase
